# Crush-A-Berry: Garden Jam!

<img src="./images/Crush-A-Berry Logo.png" alt = "Illustrated Crush-A-Berry logo featuring a jar of crushed red berries with a wooden pestle and green leaves." width = 300>

**Crush-A-Berry: Garden Jam!** in this game you'll be harvesting ripe berries, avoid rotten ones, and racing against the clock to fill your jam jar in time for a picnic. Sweet, cozy, and just a little chaotic! I like whack-a-mole as much as the next person, especially when I want to let out some healthy frustration... but maybe I feel bad for the moles! So, I made this game to crush some berries and make jam instead.
